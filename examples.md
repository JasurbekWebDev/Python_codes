## Mashq 23: Masofa, tezlik, vaqt
```python
masofa = 120 # km
tezlik = 60 # km/soat
vaqt = masofa / tezlik # soat

print(f"Masofa: {masofa} km")
print(f"Tezlik: {tezlik} km/soat")
print(f"Vaqt: {vaqt} soat")
# Natija: Vaqt: 2.0 soat
# Ushbu kod masofa, tezlik va vaqtni hisoblaydi.
# Masofa 120 km, tezlik 60 km/soat bo'lsa, vaqt 2 soat bo'ladi.
# Kodning maqsadi: berilgan masofa va tezlik asosida vaqtni hisoblash.
```
